# Go 驱动故障排查

## 常见问题

### 1. 找不到 libyascli.so / yascli.dll

**可能原因**：
- C 驱动未安装
- 环境变量未正确设置

**解决方案**：
- 执行 `/yashandb-c` 检查 C 驱动安装
- 确认 `LD_LIBRARY_PATH`（Linux）或 `PATH`（Windows）已正确设置

### 2. 连接超时

**可能原因**：
- YashanDB 服务未运行
- 端口不正确
- 防火墙阻止

**解决方案**：
- 检查 YashanDB 服务是否运行
- 检查端口是否正确（默认 1688）
- 检查防火墙设置

### 3. 认证失败

**可能原因**：
- 用户名或密码错误

**解决方案**：
- 确认用户名密码正确
